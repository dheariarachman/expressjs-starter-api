# Starterkit for ExpressJS API
    1. Express 4.x
    2. Morgan
    3. Nodemon
    4. Mongoose
    5. MongoDB
    6. BCrypt
    7. Validator

# How to Clone
    1. Clone Repo
    2. use `npm install` or `yarn install`
    3. then use `npm run start` or `yarn start`
    4. `Server Running on port 3000`
    5. Open your Postman or Insomnia
    6. Run `localhost:3000/todos`
